# Personal AI Finance Manager  

**Pakistan’s Smartest Personal Finance AI** – Built with Machine Learning & Real Pakistani Financial Data  

A complete intelligent personal finance tool that predicts your future savings, analyzes spending patterns, and gives you professional-level financial advice – all in one place.

### Features
- **Future Savings Prediction** (Next 1, 3 & 6 months) using Random Forest ML model  
- **Financial Health Score** – Excellent / Good / Average / Poor  
- **Net Worth Calculator** (Savings + Investments – Debt)  
- **Emergency Fund Status** (6-month expense rule)  
- **50/30/20 Rule Checker** with visual feedback  
- **Smart Saving Goal Planner** (Current vs Basic 20% vs Strong 30% plans)  
- **Investment Suggestion** based on your age (Aggressive / Balanced / Conservative)  
- **Beautiful Interactive Charts** (Income vs Expense, Category Spending, Trend)  
- **PDF Report Generator** – Download your full financial report  
- **Dark Mode + Clean UI** – Built with Streamlit  

### Tech Stack
- Python, Pandas, Scikit-learn  
- Random Forest Regressor (trained on real personal finance dataset)  
- Streamlit (Web App)  
- Matplotlib & Seaborn  
- ReportLab (PDF generation)
